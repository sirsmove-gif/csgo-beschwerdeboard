```markdown
# CS:GO Beschwerden – SmoVe

Einfache statische Webseite zum Sammeln von (lokal gespeicherten) Kommentaren/Beschwerden zu CS:GO. Die Kommentare werden lokal im Browser (localStorage) gespeichert — es gibt kein Server-Backend.

Features:
- Einfache, mobilefreundliche Oberfläche
- Smiley-Buttons zum Einfügen von Emojis
- Kommentare lokal mit Datum gespeichert
- Kommentare löschen / alle löschen

Deployment:
Diese Site wird über GitHub Pages bereitgestellt.
```
